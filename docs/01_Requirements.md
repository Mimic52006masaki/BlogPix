# 01_Requirements.md

## 概要
**BlogPix** は、ブログ記事内の画像を取得し、ユーザーが任意の画像を選択してPCにダウンロードできるWebアプリです。  
MVPとして、URL入力から画像取得、サムネイル表示、選択・全選択・全解除・ZIPダウンロードまでを実装対象とします。

## 機能要件
1. ユーザーがブログ記事のURLを入力できる。
2. 入力URLから記事内の画像を抽出する。
   - HTML解析で `<img>` タグを取得
   - 広告画像を除外
   - Lazy Load対応（`data-src`属性取得など）
3. 取得画像をサムネイル形式で一覧表示
4. ユーザーが複数画像を選択可能
5. 「全選択」「全解除」ボタンを提供
6. 選択画像をZIP形式でダウンロード
7. フォーマット変換機能（PNG⇄JPG）を提供

## 非機能要件
- macOS / ブラウザ上で動作
- サーバーレス構成（Vercel上で動作）
- 画像処理は軽量で高速
- UXを優先し、ページ遷移を最小化

## 仮定
- 対象ブログは一般的なHTML構造を持つブログ系サイトとする
- 最大取得画像数は50件まで（MVP想定）
