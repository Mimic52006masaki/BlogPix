# 02_UserStories.md

## ユーザーストーリー

### US01: URL入力
- **役割**: ブログ閲覧者
- **目的**: 記事内画像を取得したい
- **シナリオ**:
  1. URLを入力
  2. 「画像を取得」ボタンをクリック
  3. 画像一覧が表示される

### US02: 画像選択
- **役割**: ユーザー
- **目的**: ダウンロードしたい画像だけを選択したい
- **シナリオ**:
  1. サムネイルを確認
  2. 不要画像のチェックを外す
  3. 「全選択」「全解除」でまとめて操作可能

### US03: ダウンロード
- **役割**: ユーザー
- **目的**: 選択した画像をまとめて保存したい
- **シナリオ**:
  1. 選択画像をAPIに送信
  2. ZIP化された画像をブラウザでダウンロード

### US04: 画像変換（MVPオプション）
- **役割**: ユーザー
- **目的**: JPG⇄PNG形式に変換して保存したい
- **シナリオ**:
  1. ダウンロード前に変換形式を選択
  2. 選択画像を指定形式でZIP化
